Numbers: The Command Unit has 1-5 characters.

Weapons: All members of this squad are allow any items from the armory.

Options: All members of this squad are allow any items from the armory.

Special Rules:
Minor Setback: If Ad Infinitum are forced to fall back, then they cannot be slain in the process.
Rule 5: All Ad Infinitum squads can have only up to 5 members to a squad.


Command Link: In order to give a Doggon a retinue, a Vespic must be added to control a Linx, which controls the Mirv. If the Vespic is destroyed, then the Doggon becomes an independent character and the Linx stays to command the Mirv. If the Linx is destroyed then the Doggon and the Vespic are a squad and the Mirv become a squad. If the Doggon or a Mirv is destroyed, then the Command Unit has simply lost a member. Note: You don't have to have a Doggon for a Vespic, Vespic for a Linx, or Linx for Mirv. It is very likely to have a group of Mirv for an HQ choice.
